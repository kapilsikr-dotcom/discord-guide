Інструкція Discord
Встановлення
Завантажити Discord з офіційного сайту.
Встановити програму на комп'ютер.
Запустити Discord.
Реєстрація
Натиснути кнопку Register.
Ввести пошту та пароль.
Підтвердити реєстрацію через електронну пошту.
(Запам’ятати пароль для подальшого входу)
Створення сервера
Натиснути кнопку "+" зліва.
Обрати Create Server.
Ввести назву сервера.
Натиснути Create.
Додавання друзів
Відкрити вкладку Friends.
Натиснути Add Friend.
Ввести ім’я користувача.
Надсилання повідомлень
Обрати друга або сервер.
Ввести повідомлення у поле для тексту.
Натиснути Enter для надсилання.
GitHub репозиторій
Створити репозиторій на GitHub.
Додати файл README.md.
Виконати commit змін.
Завантажити проєкт у GitHub репозиторій.
